# Cisco Small Office Network Project

## Description
This project demonstrates a small office network setup using Cisco devices. It includes network design, configuration, and basic connectivity for a small office environment.

## Features
- Router and switch configuration
- IP addressing and subnetting
- VLAN setup
- Basic network security settings
- Connectivity testing

## Project Structure
- configs/ → Cisco device configuration files
- diagrams/ → Network diagrams
- scripts/ → Any automation or helper scripts
- README.md → This file

## How to Use
1. Open Cisco Packet Tracer or your preferred Cisco simulation software.
2. Load the configuration files from the configs/ folder.
3. Apply the configurations to the devices.
4. Test connectivity between devices.
